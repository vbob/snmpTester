# SNMP Tester
See parsed SNMP responses on the browser

![](img/print.jpeg)

## Installation

### Prerequisites

- [Node.js](http://nodejs.org)
- [net-snmp](http://www.net-snmp.org/)

Linux:

```sh
npm install
npm run build
```

## Usage 

```sh
npm start
```

On browser, access: http://127.0.0.1:8080
